Se trabajo en php, con una base de datos mysql, se utilizo el paquete WAMP, 
que en windows incluye una carpeta c:/wamp/www donde se debe descargar la carpeta GPI4 para que 
el servidor sea capaz de reconocer el proyecto.

El proyecto cuenta con un login para personal de obras y de bodega, con funciones distintas respectivas a sus cargos:

- para obras   Usuario: Obras    Contraseña:1234
- para bodega   Usuario: Bodega    Contraseña:1234
